#  HomeMade Chocolates Website

## How to Add Your Chocolate Images:

1. Save your chocolate photos in the `images` folder
2. Name them: chocolate1.jpg, chocolate2.jpg, etc.
3. Open products.html in a text editor
4. Find the comment: <!-- Replace with: <img src="images/chocolate1.jpg"... -->
5. Replace the emoji with the img tag

## Example:
Replace this:
```html
<span style="font-size: 4rem;"></span>
```

With this:
```html
<img src="images/chocolate1.jpg" style="width: 100%; height: 200px; object-fit: cover; border-radius: 10px;">
```

## To Update Prices:
1. Open products.html
2. Find: <p class="price">₹150</p>
3. Change the price number
